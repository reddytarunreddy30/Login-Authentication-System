#  Login Authentication System (Java Full Stack)

>  Built as a full-stack authentication system demonstrating real-world backend integration using Java, JSP, Servlets, JDBC, and Oracle Database

##  Overview

This project is a **Login Authentication System** developed using Java full-stack technologies. It validates user credentials by connecting to an Oracle database and ensures secure authentication using JDBC and PreparedStatements.



##  Features

-  User Login Authentication
-  JDBC-based database connectivity
-  Secure SQL queries using PreparedStatement
-  MVC-inspired structure (DAO pattern)
-  Integration with Oracle XE Database
-  Lightweight and easy to deploy on Apache Tomcat


##  Tech Stack

| Layer       | Technology Used              |
|------------|-----------------------------|
| Frontend    | HTML, JSP                   |
| Backend     | Java (JSP, Servlets)        |
| Database    | Oracle XE                   |
| Server      | Apache Tomcat               |
| Connectivity| JDBC (ojdbc11.jar)          |
